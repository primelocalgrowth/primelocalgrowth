# Quick: Get Your Beehiiv Embed Code (2 minutes)

## Step-by-Step

1. **Go to:** https://app.beehiiv.com
2. **In left sidebar:** Click "Settings" or look for "Subscription Forms"
3. **Find your form:** Look for "Local Edge" or your main newsletter form
4. **Click:** "Embed" or "Get Embed Code" button
5. **Look for the code** that looks like:
   ```
   <script async src="https://beehiiv.com/embed.js"></script>
   <div data-beehiiv-form="FORM_ID_HERE"></div>
   ```
6. **Copy the ENTIRE code** above (both lines)
7. **Return here and paste** the code in Slack/Discord or reply with it

## What You're Looking For

The code will have a **Form ID** that looks like one of these:
- `abc123def456` (numbers and letters)
- `local-edge` (words with hyphens)
- A long string like `5f8a9b2c-3d4e-5f6g-7h8i-9j0k1l2m3n4o`

**The important part:** `data-beehiiv-form="THIS_PART"`
