# Prime Local Growth - Google Workspace Organization Guide

## Master Folder Structure

This guide provides the complete folder structure for organizing your Google Workspace for maximum efficiency and automation. Follow this structure exactly to ensure all automation systems work correctly.

---

## Root Folder Structure

Create a main folder called **"Prime Local Growth - Master"** in your Google Drive. Inside, create these 7 master folders:

```
Prime Local Growth - Master/
├── 00_MASTER_CLIENT_DATABASE
├── 01_RESEARCH_LIBRARY
├── 02_SYSTEMS_FRAMEWORK
├── 03_CLIENT_PROJECTS
├── 04_TEMPLATES
├── 05_CONTENT_LIBRARY
└── 06_OPERATIONS
```

## Setup Instructions

### Step 1: Create Master Folder

1. Open Google Drive
2. Click "New" → "Folder"
3. Name it: `Prime Local Growth - Master`
4. Note the folder ID from the URL

### Step 2: Create Subfolders

1. Open the Master folder
2. Create each of the 7 main folders listed above
3. Use exact naming convention (00_, 01_, etc.)

### Step 3: Create Templates

1. Open Folder 4 (Templates)
2. Create all template documents
3. Use consistent formatting
4. Add instructions and placeholders

### Step 4: Set Up Client Projects

1. Open Folder 3 (Client Projects)
2. For each new client, create a folder using the naming convention
3. Create all subfolders as shown above
4. Copy relevant templates into the client folder

### Step 5: Configure Sharing

**Master Folder:** Shared with yourself only (private)
**Client Folders:** Shared with client (read-only or edit as appropriate)
**Templates:** Shared with yourself only
**Research Library:** Shared with yourself only
**Operations:** Shared with yourself only

### Step 6: Set Up Automation

1. Create Google Apps Script for folder automation
2. Set up triggers for automatic updates
3. Configure email notifications
4. Test all automation

## Naming Conventions

**Consistency is critical for automation and efficiency.**

### Client Folder Names
Format: `[Client Business Name] - [YYYY-MM-DD]`
Example: `ABC Plumbing - 2026-04-01`

### Document Names
Format: `[Document Type] - [Client Name] - [YYYY-MM-DD]`
Example: `Discovery Report - ABC Plumbing - 2026-04-01`

### File Names
Use clear, descriptive names with dates
Example: `Market Analysis - ABC Plumbing - 2026-04-01.pdf`

### Folder Names
Use consistent prefixes for sorting
Example: `01_DISCOVERY`, `02_ARCHITECTURE`, etc.
